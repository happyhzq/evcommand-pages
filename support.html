Common Topics

Account: use Sign in with Apple to create or access an EV Command account. You can sign out or permanently delete the account from the Account screen.

Tesla connection: sign in to EV Command first, then authorize optional read-only Fleet API access from Settings. Disconnecting removes EV Command's stored Tesla credentials and cached vehicle snapshot.

Language: choose English, Simplified Chinese, Cantonese, Korean, Japanese or Spanish from Settings.

Purchases: use the Membership screen to buy or restore the one-time EV Command Pro Lifetime unlock through Apple In-App Purchase. Deleting an EV Command account does not remove Apple-managed purchases.

Vehicle data: live, cached, stale, unavailable and demo states are labeled separately. Demo data does not require a Tesla account and never represents your vehicle.

Privacy: review the Privacy Policy for collected fields, security and deletion details.
